## 프로젝트 소개

* 프로젝트명: 애니타임
* 서비스 특징: 비대면 유기동물 입양 플랫폼
* 주요 기능
  - 회원 관리(Kakao SNS인증)
  - 화상 미팅룸(Openvidu)
  - 1 대 1 채팅
  - 추천 시스템
  - 실시간 데이터 동기화(스케쥴러를 사용한 데이터 크롤링)
  - 후원 기능(Toss 결제 API)
  - 지도를 통해 입력받는 위치정보(Kakao Map API)
  - 지도 시각화로 통계 제공
* 주요 기술
  - WebRTC
  - WebSocket
  - RabbitMQ
  - JWT Authentication
  - REST API
  - JPA
  - OpenAPI
  - Schedule
* 참조 리소스
  * Mui : 버튼 및 모달 등 컴포넌트 사용
  * Openvidu : Openvidu 서버 사용
  * Kakao API : SNS 로그인 및 지도 API 사용
* 배포 환경
